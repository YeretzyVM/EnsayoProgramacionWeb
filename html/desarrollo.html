Servidor Web

Vamos a comenzar mencionando la importancia de contar con un servidor web, lo primero que debemos tener en cuenta es que gracias a los servidores podemos hacer peticiones de servicios, como mostrarnos páginas de internet, fotos, imágenes, videos, documentos o alguna función que solicite un cliente.
Un servidor web puede ser: el software del servidor, es decir, un programa en sí, o bien, puede ser al equipo en el que el software web se está ejecutando, lo cual es mejor conocido como host, también conocido como hosting, es cualquier computadora o máquina conectada a una red mediante una IP ya preestablecida y un dominio, este ofrece recursos, información y servicios a sus usuarios. Un ejemplo de un servicio de host es internet, que conecta dispositivos como computadoras personales, servidores, etc., además ofrece la garantía de que, en caso de presentarse algún problema en un servidor, entrará uno más que se encargará de inmediato a cubrir tal problema.
Es por ello que antes de crear un proyecto en el que se sabe que se ocupará un servidor web, tenemos que ver cuál será el correcto a elegir, debido a que debemos contar con el más fiable ya que si nuestro sitio web deja de funcionar, debe volver a la red de inmediato para que nuestro proyecto o negocio no se vean afectados, es por ello que un servidor debe estar conectado a internet todo el día.
Por lo que es de suma importancia fijarnos bien que servidor es el correcto, de hecho, se recomienda que los servidores aseguren esta garantía, además de realizar copias de seguridad del proyecto.
Como mencioné al inicio se puede tener como servidor un host de internet, o bien, podemos ser nuestro propio host en nuestra propia red, por ejemplo, un enrutador responsable de una red de varias computadoras que se conectan entre sí a través de una IP también puede considerarse un host.
Adentrémonos un poco a cómo es que funciona un servidor web, cada vez que abrimos el navegador y tecleamos una dirección url, se establece una conexión con el servidor en el que se encuentra el sitio, de esta forma podemos acceder a él. Mediante este proceso, es como el nombre de nuestro sitio web se convierte en una dirección IP y de esta manera se consulta la información que requerimos.
Por ejemplo, un sitio web de moda, muestra el siguiente nombre de dominio, es decir su link o url, por ejemplo, Forever21.com, está conectado a un servidor a través del IP 120.200.80.32, la cual es la IP que requiere el cliente para conectarse al dominio mediante al servidor. Cabe mencionar que cada dominio tiene una IP por lo que siempre son distintas, es imposible tener dos sitios diferentes con las mismas url, ya que esto haría una colisión entre páginas.

Plataformas Front-End Y Back-End

Para poder diseñar nuestras plataformas web, es necesario contar con diferentes opciones de lenguajes a elegir, tomando en cuenta que existen los leguajes que ayudan a desarrollar la interfaz del usuario (Front-End) así como los que ayudan a programar el cerebro de la página web (Back-End), entre los mas destacados para trabajar con la interfaz de las páginas web, y extrayendo esta información de una fuente muy correcta y precisa, el cual es un gráfico que se puede encontrar en el curso de la fundación Carlos Slim, en donde se puede certificar en algún curso de su agrado, en este curso se encuentra la información de los mejores lenguajes para el desarrollo web, los cuales son: HTML5, JavaScript y CSS, los cuales se encargan de el diseño que el usuario ve, es decir, iconos, textos, banner, colores, imágenes, videos, anuncios, etc., también podemos encontrar lenguajes que son más orientados a el Back-End, es decir, al cerebro de la página web, entre los que más destacan son: Java, PHP, Python, C# y Ruby.
Así como es primordial elegir el servidor que mejor se adapte a los requisitos y objetivos que tenemos que lograr con el desarrollo de la página web, es necesario también saber que herramientas de desarrollo como el lenguaje, hay que elegir, tomando en cuenta que muchos lenguajes son más intuitivos, fáciles, lógicos, visuales, etc.

Bases De Datos

Al desarrollar una página web, proyecto o aplicación, tenemos que tomar en cuenta que también puede necesitar una base de datos, ya que los usos son primordiales, desde registrar sesiones, usuarios y contraseñas con las que los usuarios acceden a la plataforma, hasta elaborar un traductor en donde contengamos bases de datos con las palabras más comunes del español, traducidas a los idiomas a elegir, o bien, las bases de datos son primordiales en aplicaciones de gestión de datos, tales como las bases de datos de un banco, los cuales gestionan el dinero de cada persona, desde transacciones, depósitos, pagos, intereses, créditos, prestamos, etc.


Versiones y Compatibilidad

Al crear una plataforma, o bien, al comenzar a desarrollarla, tenemos que tener en cuenta que los lenguajes o plataformas a utilizar pueden no ser las más convenientes o bien, pueden ser no tan comunes, que incluso algunos usuarios puedan no ver la información, o en caso contrario, verla de una manera diferente a la que se pensó y estableció. Por ejemplo, hay plataformas en internet que no se muestran tal cual se muestran en alguna computadora de escritorio, o no muestran los mismos resultados, como es el caso de la plataforma de ventas Amazon.com, en donde los resultados de búsqueda son muy diferentes al consultarlo en computadoras de escritorio o desde un teléfono celular. Por lo que cada diseño, color, tipo de fuente, lenguaje de programación, tamaño de títulos, banners, imagen, vídeos o cualquier recurso que se utilice en una plataforma web, debe estar muy pensado para ocuparse por distintos usuarios, en diferentes tipos de pc, diferentes sistemas operativos, o bien, diferentes dispositivos, por lo que se debe asegurar que las versiones del software, SO o la compatibilidad de nuestro software debe ser de compatibilidad.
